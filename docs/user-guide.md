# Magus Cloud 用户手册

![登录页](/help-assets/login-screen.png)

## 1. 使用目的

本系统用于内部文件存储、文件查询、在线预览、文件分享与个人空间管理。

## 2. 登录系统

1. 访问系统首页。
2. 点击登录入口。
3. 使用账号密码或飞书登录。
4. 登录成功后进入文件工作台。

## 3. 文件工作台

![文件工作台](/help-assets/dashboard-screen.png)

登录后可执行以下操作：

- 查询文件与目录
- 上传文件
- 上传文件夹
- 创建目录
- 下载文件
- 删除文件
- 打开文件预览
- 创建分享链接
- 查看个人空间使用量

## 4. 文件操作说明

### 上传文件

1. 进入目标目录。
2. 点击上传文件。
3. 选择本地文件并提交。

### 上传文件夹

1. 进入目标目录。
2. 点击上传文件夹。
3. 选择本地文件夹并提交。

### 创建目录

1. 进入目标位置。
2. 点击新建目录。
3. 输入目录名称并确认。

### 下载文件

1. 在文件列表中选中目标文件。
2. 点击下载。

### 删除文件

1. 在文件列表中选中目标文件或目录。
2. 点击删除并确认。

## 5. 文件预览

系统支持以下文件类型预览：

- 文本文件
- 图片文件
- PDF 文件
- Word 文件
- Excel 文件
- PowerPoint 文件

## 6. 文件分享

1. 在文件列表中选择目标文件。
2. 点击分享。
3. 设置分享参数。
4. 复制分享链接并发送。
5. 在分享列表中查看或删除分享记录。

## 7. 帮助入口

![帮助抽屉](/help-assets/help-drawer.png)

系统页面右上角提供帮助入口，可随时重新打开本手册。
