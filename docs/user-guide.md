# Magus Cloud 用户文档

![登录页预览](/help-assets/login-screen.png)

## 1. 快速开始

Magus Cloud 是一个面向团队文件协作的云盘工作台。普通用户最常使用的能力有：

- 登录并进入文件工作台
- 上传文件或整个文件夹
- 在线预览常见文档和图片
- 创建分享链接并发送给其他人

## 2. 登录

1. 打开首页
2. 点击首页中的“去登录”
3. 在弹出的登录层中选择“使用飞书登录”
4. 登录成功后自动进入 `/dashboard`

管理员应急登录入口位于登录层折叠区，仅用于初始化、排障和配置维护。

## 3. 文件工作台

![工作台预览](/help-assets/dashboard-screen.png)

进入工作台后，你会看到：

- 一级侧栏：首页、文件、分享、回收站、帮助
- 二级侧栏：我的文件、图片、文档、视频、其他、回收站、快捷访问
- 顶部工具条：上传文件、上传文件夹、新建文件夹、刷新、文件/分享切换、搜索框
- 主内容区：文件表格、分享记录、容量状态卡片

## 4. 上传与下载

### 上传文件

点击“上传文件”，选择单个文件后系统会直接上传到你的用户目录。

### 上传文件夹

点击“上传文件夹”，选择整个目录后会保留原始目录结构。

### 下载

在文件表格中点击“下载”即可：

- 文件会直接下载原文件
- 文件夹会打包为 zip 下载

## 5. 预览

系统支持常见文本、图片、PDF、Office 文档的在线预览。

- 文本、代码、JSON：直接文本预览
- 图片：原图预览
- PDF：PDF 预览
- Word / Excel / PowerPoint：按当前环境可用能力转换预览

## 6. 分享

1. 选择文件或文件夹
2. 点击“分享”
3. 设置有效期
4. 复制生成的分享链接和提取码

分享页支持：

- 直接下载
- 提取码访问
- 到期自动失效

## 7. 配额与空间

你可以在工作台顶部看到：

- 个人已用空间
- 个人配额
- 全局已用空间
- 可分配空间

如果上传被拒绝，通常是因为：

- 你的个人配额已满
- 当前总可分配空间不足

## 8. 当前版本说明

- 回收站入口已预留，但当前删除仍为直接删除
- 容量策略字段会展示在界面中，但不改变当前真实上传限制

## 9. 获取帮助

- 页面右上角 `?` 可以随时重新打开这份文档。
- 如果管理员配置了帮助链接，也可以从界面中直接跳转。
