# Magus Cloud 管理员手册

![管理后台](/help-assets/admin-screen.png)

## 1. 管理职责

管理员负责系统配置维护、节点与存储状态查看、用户配置校验、备份与迁移操作、运行日志查看。

## 2. 登录后台

1. 进入系统首页。
2. 打开登录入口。
3. 使用管理员账号登录。
4. 登录成功后进入管理后台。

## 3. 后台功能

管理员后台提供以下功能：

- 系统设置查询与更新
- 服务配置查询与更新
- 云配置查询与更新
- 节点状态查看
- 存储状态查看
- 备份列表查看
- 运行日志查看
- 站点快照创建
- 用户导出
- 用户导入

## 4. 配置维护

系统主要配置文件如下：

- `config/magus.config.json`
- `config/cloud.config.json`

后台变更配置后，系统会将结果写回配置文件。

## 5. 巡检内容

管理员应定期核查以下内容：

- `/api/health`
- `/api/health/gateway`
- 节点在线状态
- 存储可用状态
- 备份生成情况
- 日志异常信息

## 6. 备份与迁移

### 站点快照

管理员可发起站点快照，系统将生成归档文件和清单文件。

### 用户导出

管理员可按用户名导出单个用户文件与分享数据。

### 用户导入

管理员可将归档文件导入到目标用户名目录，并支持 Dry Run 方式预检。

## 7. 帮助入口

管理员后台右上角提供帮助入口，可随时重新打开本手册。
